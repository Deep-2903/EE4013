\section{Problem}
Given the following matrix we have to find the largest eigen value.
$\operatorname{M}$=$\left[\begin{array}{llll}
0 & 1 & 1 & 1 \\
1 & 0 & 1 & 1 \\
1 & 1 & 0 & 1 \\
1 & 1 & 1 & 0
\end{array}\right]$

\section{Solution}
For this we can directly find the eigen value by using determinant method (Here I is identity matrix) i.e.,
\begin{align}
    |M - \lambda*I| = 0
\end{align}
The above equation will give the following equation,
\begin{align}
    ($\lambda$ - 3)($\lambda$+1)^{3} = 0
\end{align}
From this we can see that the roots are $\lambda$ = 3, $\lambda$ = -1, $\lambda$ = -1 and $\lambda$ = -1. These are the eigen values and the biggest one is 3. So the answer is 3.
\newline
\newline
The other method is using Power method which is done in C language as well. The code for it is already shared at the start of this pdf. In this method we will be assuming a column vector
$\operatorname{X}$=$\left[\begin{array}{llll}
1 \\
1 \\
1 \\
1
\end{array}\right]$
such that 
\begin{align}
    MX - \lambda X = 0
\end{align}

After running the code accordingly we should get the largest eigen value as 3.
